National Geographic Article


A project to help with the understaing of how to layout a lot of different types of information on a single webpage.


This project was a difficult one, I struggled with how to make so much text on a page digestible, how to break it up but still keep it cohesive. I went with sections with the text moving from left to right down the page. This with the use of images throughout I think really helps break the infomation up into more manageable chunks.


This project was also about macro and micro interactions, I had a lot of fun with this part; what does a button do when you hover over it? How should the page load in? There were a lot of things to consider and it taught me a lot about the types of animations you can do using only css and what you have to create using Javascript.
